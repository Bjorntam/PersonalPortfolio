## Description
ValACE Project | Solo Development | Completed

A multi-game Android educational platform designed and developed for children in Valenzuela City, featuring interactive learning-focused mini-games.

- Developed using **Unity** (Android) and **Retool** (Web App); integrated **Firebase** for database.
- **Role: Lead Game Developer** - Developed core gameplay systems, implemented interactive mechanics, programmed game logic, and designed UI/UX elements
