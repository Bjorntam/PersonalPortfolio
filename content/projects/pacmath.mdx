## Description
Commissioned Project | Solo Development | Completed

A mobile educational game designed to teach basic addition to Grade 1 students. It features an endless mini-game where a Pac-Man–style character jumps over ghosts. Players can customize the character’s hat after completing all difficulty levels. The game also includes a leaderboard system.

- Developed in **Unity** with UI design in **Figma**; integrated **Firebase** for backend services and **Retool** for student database front-end dashboard.
