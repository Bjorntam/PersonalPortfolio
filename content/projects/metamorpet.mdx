## Description
Capstone Project | Group Development | Completed

MetamorPET is a gamified Android application developed to help preschool students of Metamorphosis School Inc. learn and manage their daily routines in a fun and engaging way. The app features a virtual pet that students care for by completing validated routines, promoting responsibility and consistency. By integrating educational goals with interactive game elements, MetamorPET aims to enhance learning, support parental involvement, and streamline school monitoring through a user-friendly system.

- **Designed and developed in Unity:** (Android) and React + Vite + TypeScript + Tailwind CSS (Web App); integrated Firebase for database and authentication.
